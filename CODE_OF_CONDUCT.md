# Code of Conduct

## Our Pledge

Nous nous engageons à faire de la participation à ce projet une expérience sans harcèlement pour tous.

## Our Standards

Exemples de comportements attendus :
- Être respectueux et inclusif
- Accepter les critiques constructives
- Faire preuve d’empathie envers les autres contributeurs

## Enforcement Responsibilities

Les mainteneurs du projet sont responsables de clarifier et d’appliquer les standards de comportement acceptables.

## Scope

Ce code de conduite s’applique dans tous les espaces du projet.

## Enforcement

Les comportements inacceptables peuvent être signalés aux mainteneurs du projet.

## Enforcement Guidelines

Les mainteneurs suivront des lignes directrices équitables pour traiter les violations.

## Attribution

Ce code de conduite est adapté du Contributor Covenant.
